Shapes

Shape classes

OSGL provides a set of shape classes that represent simple common shape entities. Each shape is different, but they all have the same format in the argument buffer. Every one of the primitives below use the shape class. Like any other entity, you can pass the shape through draw to render it.

Circle

You can use the circle function to create a circle. It requires a color, position, and radius:
shape.circle({ Color = color.Blue, Position = Vector2.new(50, 50), Radius = 5 })

Rectangle

You can use the rectangle function to construct a rectangle. Like other primitives, it requires a size, position, and color:
shape.rectangle({ Color = color.Blue, Position = Vector2.new(50, 50), Size = Vector2.new(50, 50), })

Triangle

You can use the triangle function to create a.. well triangle.. It only requires a color, and 3 points (Vector2). Note: This function calls shape.convex. It is the same as passing 3 arguments through that function.
shape.triangle({ Color = color.Blue, P1 = Vector2.new(10, 10), P2 = Vector2.new(100, 10), P3 = Vector2.new(100, 100), })

Convex

A convex shape is a custom shape with any amount of vertices allowed.
shape.convex({ Color = color.Blue, Verts = {Vector2} })

Regular Polygon

Allows for the creation of a regular polygon.
shape.polygon({ Color = color.Blue, Sides = 5, --// Pentagon Position = Vector2.new(50, 50), Size = Vector2.new(25, 25) })

Line

Draws a line from P1 to P2.
shape.line({ Color = color.Blue, P1 = Vector2.zero, P2 = Vector2.one, Thickness = 3 })


You can also create your own shapes and rendering methods with: vertex arrays.
